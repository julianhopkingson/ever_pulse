# 定时鼠标移动工具 (Mouse Mover)

[English Documentation](README.md)

一个轻量级的 Windows 工具，通过模拟微小的鼠标移动来防止电脑进入休眠或锁屏状态。

![软件预览](doc/v1.0_ui_preview.png)

## 功能特点

- **防休眠/防锁屏**：保持 Teams、Slack 等办公软件的“在线”状态。
- **智能空闲检测**：仅在您离开电脑（空闲）时运行。一旦检测到您在使用鼠标或键盘，工具会自动暂停干扰。
- **定时运行**：支持设置具体的“开始时间”和“结束时间”。工具会在工作时间自动运行，下班后自动停止。
- **无感运行**：鼠标移动后会立即复位（抖动模式），在视觉上几乎察觉不到，不影响正常工作。
- **多语言支持**：内置 **中文** 和 **English** 界面，可一键切换。
- **零依赖**：基于 Python 标准库 (`tkinter`, `ctypes`) 构建，无需复杂的环境配置。

## 使用方法

1.  确保您的电脑已安装 Python。
2.  运行脚本：
    ```bash
    python mouse_mover.py
    ```
3.  设置您的 **开始时间** 和 **结束时间**。
4.  点击 **开始**。

## 配置说明

- **间隔时间**: 鼠标移动的频率（秒）。
- **空闲时间**: 触发自动移动所需的空闲时长（秒）。
- **移动方向/像素**: 自定义鼠标移动的方向和距离。
