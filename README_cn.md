# 恒动 (Ever Pulse)

[English Documentation](README.md)

**v2.2 (恒动极客版)**

一款现代化、高颜值的 Windows 效率工具，防止电脑进入锁屏或休眠状态。采用 **PySide6** 重构，带来绝美的磨砂玻璃视觉体验。

![软件预览](assets/v2.0_ui_preview_cn.png)

*(注：v2.2 界面已升级为 恒动 品牌风格)*

## 功能特点

- **极致 UI**: "Crystal Green" 水晶绿设计，支持实时日/夜间模式切换与丝滑动画。
- **智能空闲检测**: 仅在您离开电脑时（空闲超过阈值）自动介入。
- **定时运行**: 支持设置上下班时间，自动启停。
- **隐形微动**: 鼠标仅作微小抖动并立即复位（Jiggle），肉眼几乎无法察觉，但足以保持在线状态。
- **双语支持**: 内置 **中文** 和 **English**，界面一键切换。
- **便携与持久**: 单个 EXE 文件，设置永久保存至 `config/config.ini`。
- **高稳定性**: 采用 ARGB Hex 渲染，彻底解决黑屏与闪退问题。

## 使用方法

1.  **直接运行**: 双击 `dist/ever_pulse.exe`。
    *(注意: 如需更新，请先运行 `taskkill /F /IM ever_pulse.exe` 关闭现有进程)*

2.  **源码构建**:
    ```bash
    # 1. 安装项目依赖
    pip install -r requirements.txt
    
    # 2. 打包生成 EXE (清理缓存并自动确认)
    pyinstaller main.spec --clean --noconfirm
    ```
3.  设置您的 **开始时间** 和 **结束时间**。
4.  点击 **开始**。

## 配置说明

> **注意**: 配置文件会自动生成在 `config/config.ini`。

- **间隔时间**: 鼠标移动的频率（秒）。
- **空闲时间**: 触发自动移动所需的空闲时长（秒）。
- **移动方向/像素**: 自定义鼠标移动的方向和距离。
